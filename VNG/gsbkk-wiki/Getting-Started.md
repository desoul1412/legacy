# Getting Started

## Prerequisites

- Python 3.8+
- Apache Spark
- Git

## Installation

1. Clone the repository:
   ```bash
   git clone https://code.vng.vn/baoln2/gsbkk.git
   cd gsbkk
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
